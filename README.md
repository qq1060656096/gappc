# gappc
> gappc 项目 创建gin框架脚手架, 快速创建web项目

```
go get -u github.com/qq1060656096/gappc
go install github.com/qq1060656096/gappc
```

```sh
$ gappc -h
NAME:
   gappc - gapp command

USAGE:
   gappc [global options] command [command options] [arguments...]

VERSION:
   0.0.1

AUTHOR:
   andy <1060656096@qq.com>

COMMANDS:

     api         apiName  创建api
     app         appName  创建app
     middleware  middlewareName  创建中间件
     new         projectName 创建项目
     theme       themeName subDirName fileName 创建主题文件
     help, h     Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version
```
